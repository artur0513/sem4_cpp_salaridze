Если использовать другие лямбда функции test_lambda, то разность времени между
параллельным и последовательным вариантом может быть больше. Я видел отличие
раз в 10-11 на 12-ти поточном процессоре.

Можно проверить что функция выполняется один раз для каждого элемента, например 
я нашел такой простой вариант - использовать такую лямбда функцию:
 
auto test_lambda = [](int& a) {
	a++;
};

Каждый элемент массива будет увеличен на 1. Далее проще всего подсчитать сумму 
элементов массива до и после выполнения for_each, она должна быть в точности равна
размеру массива. 
Или сохранить изначальный массив и для каждого элемента проверить что он изменился на 1.
